# Data Package README

This package is central to GpxAnalyzer's data management. It encompasses:

*   Data models (POJOs)
*   Data Access Objects (DAOs) for local database (Room)
*   Repositories to abstract data sources
*   Network service interfaces (Retrofit)
*   Mappers for data transformation
*   Caching mechanisms
*   Specialized sub-packages for GPX data processing (raw, dsp, cumulative, extrema, statistics).

For a comprehensive overview of all components, their roles, and the data flow architecture, please see `explanation.md`.
